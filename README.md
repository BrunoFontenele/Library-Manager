Projeto de Programação com Objetos. Primeiro período do segundo ano de Engenharia Informática e de Computadores do ano de 2025/2026 no Instituto Superior Técnico. 

Para compilar: javac -cp po-uilib.jar:. `find bci -name "*.java"`
para executar: java -cp po-uilib.jar:. bci.app.App
para executar uma biblioteca já existence: java -Dimport=ficheiro.im -cp po-uilib.jar:. bci.app.App
